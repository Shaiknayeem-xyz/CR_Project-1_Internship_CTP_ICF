# CR_Project-1_Internship_CTP_ICF
Study ASP-P1-001 is a 28-day Phase I trial assessing the safety of daily low-dose aspirin in 20 healthy adults. It monitors adverse events like GI irritation and bleeding. By inhibiting COX-1, the drug reduces thromboxane $A_{2}$ to limit platelet aggregation. The study follows ICH-GCP ethical standards
